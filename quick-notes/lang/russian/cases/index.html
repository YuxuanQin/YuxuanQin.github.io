俄语属格（Genitive Case）变格规则

名词变格

词性 单数规则 单数例子 复数规则 复数例子
阳性 -а/-я
(辅音结尾加-а，-й/-ь结尾加-я)		 стол → стола
музей → музея
учитель → учителя		 -ов/-ев/-ей
(硬辅音加-ов，-й/-ь/-ц加-ев，其他加-ей)		 столы → столов
музеи → музеев
учители → учителей
阴性 -ы/-и
(硬辅音后加-ы，-а→-ы；软辅音后加-и，-я→-и)		 книга → книги
неделя → недели		 去掉-а/-я加-ь/-ей
(-а→∅，-я→-ь/-ей)		 книги → книг
недели → недель
中性 -а/-я
(-о→-а，-е→-я)		 окно → окна
море → моря		 去掉-о/-е加-ь/-ей
(-о→∅，-е→-ь/-ей)		 окна → окон
моря → морей

词性 单数阳性 单数阴性 单数中性 复数

  1. 硬变化形容词 (новый):

    • 单数：синего (м.), синей (ж.), синего (ср.)
    • 复数：синих
