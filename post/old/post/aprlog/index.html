2021-04-02 随记

​ 难得又有机会写一篇随记, 三月月考考得很差, 上周回家都没怎么用电脑, 这周趁着放清明假更新一下.

​ Edge 的插件「IGG 谷歌访问助手」彻底无法访问了, 甚至已经被 Edge 插件商店给下架了, 其实挺好用的, 可惜矣. 不过说来也巧, 周四电脑课之后姚皇告诉我 SSRr / SSR 可以用, 今天就去 Github 和 Telegram 试着找了一下了一下[1], 下载下来之后发现居然真的有用, 至少上 Google 没问题了, 维基大典也可以正常访问了 (不用看万维的广告真好).

​ 上周帮十字街修改了下 新人指南, 实际上主要内容都是 Arwen 撰写的, 不过排版实在是一言难尽 …… 站长说高考之后做一次大更新, 他也表示过要加入私聊功能和移除 LaTeX 的决定我很不喜欢. 对数学公式的支持是一个聊天室或者即时通讯软件的加分项, 移除公式并不是一个好决定.

​ 以前十字街有人说「某岛」云者, 一直不知道是啥玩意, 一查发现是 AcFun 之前的管理者建立的论坛——A 岛; 据说还有过「沉岛」事件, 这样一个小众的论坛居然还有 UWP 实在难得, 之前一直上不去主岛, 只能上备胎岛, 上周突然就可以了, 这段时间逛了一下发现还是挺有意思的, 尤其是里面的「姐妹 1」板块 (°∀。)[2].

​ 学校期中考之后要开足球赛, 名曰「校长杯」——结果被牪牪阳称为「穿林打叶」杯. 和自己班的人一起踢感觉会好很多, 配合之类的也应该会比其他队默契一些吧 😎. 总之还是希望能拿个前一点的名次.

​ 物理考试得了 93 分 (不犯傻可以 100 的!); 算学考试 89 分, 选择题扣了 20 分就很不爽. 下次得小心一点, 算学必须要考好些才对得起数竞生的称号!

​ 去图书馆借阅了李贽所著《藏书》, 目前看到光武帝. 古书还是更适合竖排繁体旧字形, 而且此书之注也很好, 有一种「有人陪我一起在看」的感觉. 目前仔细看了项羽、刘邦和王莽传, 最敬佩的当然还是项王了[3].

​ 有人[4] 推荐了一个可以进行公式交流的即时通讯应用 Element, 但是我用 SSR 进不去, 用小手电却可以.
